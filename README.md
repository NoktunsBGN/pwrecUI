# pwrecUI - A simple GTK3 GUI for pw-record using Python 3

## Purpose

This script simply acts as a GTK3 user interface for pw-record. I was getting annoyed with complex and bloated recording tools like Audacity - those are great when you need complexity, but I wanted a very simple voice recording script so I could make small recordings to send to friends and such. So here it is. Obviously requires Python3, GTK3, and pw-record. Probably won't work if you're not using pipewire. But it's what I'm using so this is what I made it with.

Anyway, this is a very simple script, you could probably make something similar (and better) yourself, but if you don't wanna bother, this one is perfectly functional (only tested on Manjaro with KDE Plasma).

## Features

AMAZING features such as:
- Default output file naming having the date and time of the recording, as well as a randomized number so filenames don't overlap
- A live recording length timer (wow!)
- A totally awesome message at the end showing you the output path and the final recording length and file size
